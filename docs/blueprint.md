# **App Name**: Aura Study Tracker

## Core Features:

- Focus Statistics Dashboard: Visual tracking of 'Daily Focus Score' via a radial progress circle, 'Daily Streak' counters, and aggregated hours using persistent local storage.
- Custom Pomodoro Controller: Interactive high-precision timer with Start, Pause, and Reset functionality, including support for custom user-defined study session lengths.
- Automated Break Orchestrator: A smart flow tool that automatically transitions between work cycles and a 5-minute break mode with integrated audio alerts and session tracking.
- Intelligent Focus Guide: A generative AI tool that analyzes your pending study tasks and streak history to suggest an optimized focus/break schedule for the day.
- Dynamic Study Task Manager: An interactive task checklist with strike-through functionality that updates dashboard metrics in real-time as tasks are completed.
- Ambient Persistent Storage: Comprehensive integration with browser LocalStorage to ensure user progress, timers, and task lists remain consistent between sessions.
- Full-Screen Responsive Architecture: A layout optimized to consume 100% of the viewport on every device, adjusting dynamically to unique screen sizes and aspect ratios.

## Style Guidelines:

- Primary Color: Vibrant Neon Purple (#BB86FC) to evoke energy and deep concentration within a high-performance workspace.
- Background Color: Deep AMOLED Dark (#121212) for visual comfort and battery efficiency, creating a premium canvas for study tools.
- Accent Color: Luminous Teal (#03DAC6) for high-impact visual feedback, progress indicators, and highlight states.
- A font pairing of 'Poppins' for precise headlines and 'Inter' for a machined, objective feel in body text and statistical data.
- Modern outline-style icons with a glowing purple filter, emphasizing critical navigation points like the flame icon for daily streaks.
- Full-Viewport Layout: Implementation of 100vh and 100vw constraints and flexible containers to ensure the app occupies the full screen across all device sizes.
- Soft radial transitions on progress bars and subtle fading tab transitions to create a serene, flow-like state during navigation.
